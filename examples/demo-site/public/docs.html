Getting Started

Welcome to the OpenFinch Demo Store documentation. This page contains structured documentation content for testing OpenFinch's fetch and extract capabilities.

API Reference

The demo site provides the following API endpoints:

  • GET /health — Health check endpoint, returns JSON with status
  • GET /api/products — Returns all products as JSON array

Product Schema

Each product in the catalog has the following fields:

  • id (number) — Unique product identifier
  • name (string) — Product name
  • price (number) — Price in USD
  • category (string) — Product category (laptops, tablets, headphones, monitors, accessories)
  • rating (number) — Average rating from 1.0 to 5.0
  • inStock (boolean) — Whether the product is currently in stock
Tip: Use the /api/products endpoint for structured product data extraction testing.

Integration Guide

Fetching Product Data

curl -X POST http://localhost:8787/v1/fetch \
  -H 'Content-Type: application/json' \
  -d '{"url": "http://localhost:4173/products", "format": "markdown"}'

Extracting Product Information

curl -X POST http://localhost:8787/v1/extract \
  -H 'Content-Type: application/json' \
  -d '{
    "url": "http://localhost:4173/products",
    "prompt": "Extract all product names, prices, and ratings"
  }'

Browser Screenshot

curl -X POST http://localhost:8787/v1/browser/session \
  -H 'Content-Type: application/json' \
  -d '{"headless": true}'
