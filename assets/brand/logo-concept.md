# OpenFinch Logo Concept

## Core Symbol

A small, minimalist finch bird rendered as a geometric mark.
The bird faces right (forward), suggesting speed and forward motion.
The body incorporates subtle network/binary elements to signal the AI/infrastructure focus.

## Design Principles

- **Minimalist** — single-color capable, works at 16px favicon size
- **Distinctive** — not another generic bird or robot
- **Approachable** — friendly but technical
- **Timeless** — not trend-driven

## Concept Sketches

### Option A: Geometric Finch
- Clean polygon-style finch side profile
- 5-7 geometric shapes forming head, beak, body, wing, tail
- Beak slightly open as if singing/chirping
- One circular eye detail

### Option B: Finch + Browser
- Simplified finch head as the "tab" in a browser window
- Browser chrome forms the perching branch
- Implies "web browser" visually

### Option C: Finch + Network Node
- Finch body made of connected circles (network nodes)
- Wing is a directional arrow (forward/send)
- Tail forms a signal wave

## Logo Lockup

Preferred layout: Symbol + "OpenFinch" text to the right

- Font: Inter or SF Mono (sans-serif, slightly technical)
- Text weight: Semi-Bold for "OpenFinch"
- Symbol sits at same height as uppercase "O"
- Clear space: equal to the width of the symbol on all sides

## Variants Needed

1. `logo.svg` — Full lockup (symbol + text), dark background
2. `logo-light.svg` — For dark backgrounds (white symbol)
3. `mark.svg` — Symbol only (favicon)
4. `mark-icon.svg` — Simplified symbol for 16-32px sizes
5. `og-image.png` — 1200×630px social preview

## Favicon

- 32×32 SVG or ICO
- Must be recognizable at small sizes
- Use bold geometric shapes only (no thin lines < 1px)
