# GitHub Social Preview Concept

## Size
1200 × 630px (GitHub standard og:image)

## Layout

```
+------------------------------------------------------------------+
|                                                                  |
|   [Finch Symbol]   OpenFinch                                    |
|                    Open-source self-hosted AI web agent infra    |
|                                                                  |
|   Search · Fetch · Extract · Browse · Agent                     |
|   CLI · SDK · MCP · Dashboard                                   |
|                                                                  |
|   Bring your own model. Own your data. No platform fee.          |
|                                                                  |
|   [github.com/JonusNattapong/openfinch]                               |
+------------------------------------------------------------------+
```

## Design Spec

- Background: Dark slate (#0F172A) with subtle teal gradient overlay
- Top-left: OpenFinch bird mark (white/teal, ~64px)
- Next to mark: "OpenFinch" in Inter Semi-Bold, white, ~48px
- Tagline: "Open-source self-hosted AI web agent infrastructure" — white, ~24px
- Capability row: 5 pill-shaped badges — Search, Fetch, Extract, Browse, Agent — teal fill, white text
- Platform row: 4 smaller badges — CLI, SDK, MCP, Dashboard — slate fill, white text
- Bottom tagline: "Bring your own model. Own your data. No platform fee." — teal (#0D7377), ~20px
- URL: "github.com/JonusNattapong/openfinch" — white, ~16px, bottom right

## Variants

1. `github-social-preview.png` — Primary (dark)
2. `github-social-preview-light.png` — Light variant for repositories with light themes

## Notes

- No people or photos
- No gradients that look dated
- Must be readable at small sizes (GitHub displays at 640×336px)
- Text should be clearly legible when scaled down
