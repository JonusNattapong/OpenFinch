# Changelog

## [0.1.0] - 2025-05-08

### Initial Release

- API Gateway with health, search, fetch, extract, browser, and agent endpoints
- LLM provider support: OpenAI, Anthropic, Gemini, OpenRouter, Ollama
- Browser automation via Playwright (session management, screenshots)
- Agent execution with configurable steps, timeouts, and providers
- MCP server for Claude Desktop, VS Code, Cursor, and Windsurf integration
- Web dashboard for system overview and API playground
- JavaScript SDK (npm: openfinch)
- Python SDK (pip: openfinch)
- CLI tool (npx openfinch)
- Cookbook with recipes and n8n workflow templates
- Claude skill for agent-to-agent collaboration
- Docker Compose deployment with Postgres, Redis, SearXNG, MinIO
- Comprehensive documentation and safety policies
